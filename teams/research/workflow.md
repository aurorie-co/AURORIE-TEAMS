# Research Workflow

## Research Request
Trigger: open-ended research question or competitive analysis task

Steps:
1. research-lead scopes the question and defines depth of search
2. research-web gathers information from web sources
3. research-synthesizer compiles and writes the research report
4. Write output to `.claude/workspace/artifacts/research/<task-id>/`

## Quick Lookup
Trigger: factual question or narrow information need

Steps:
1. research-lead routes directly to research-web for targeted lookup
2. Write output to `.claude/workspace/artifacts/research/<task-id>/`
