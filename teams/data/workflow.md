# Data Workflow

## Analysis Request
Trigger: ad-hoc analysis question or business metrics inquiry

Steps:
1. data-lead scopes the analysis and identifies data sources
2. data-analyst queries data and produces insights
3. data-reporting formats findings into a shareable report
4. Write output to `.claude/workspace/artifacts/data/<task-id>/`

## Pipeline Development
Trigger: new data pipeline or ETL task

Steps:
1. data-lead reviews requirements and data contracts
2. data-pipeline implements the pipeline logic
3. Write output to `.claude/workspace/artifacts/data/<task-id>/`
