# v0.9.2 — Patch: Rename orchestrator → aurorie-orchestrator

> Fixes invocation collision with Claude Code's built-in orchestrator by renaming the top-level dispatcher agent.

---

## Problem

When invoking via `@orchestrator`, Claude Code's built-in orchestrator was selected instead of the project's `aurorie-teams` dispatcher. This caused routing ambiguity and incorrect agent behavior.

---

## Changes

### Rename

| `fallback: "orchestrator-clarify"` | `fallback: "aurorie-orchestrator-clarify"` |
| `@orchestrator "prompt"` | `@aurorie-orchestrator "prompt"` |
